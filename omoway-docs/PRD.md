# PRD — Omoway Homepage Redesign

## Product overview
Omoway is a Singapore-based intelligent EV brand launching the OMO X —
the world's first mass-produced self-balancing electric motorcycle.
This is the homepage for omoway.com/en targeting tech enthusiast
early adopters in Southeast Asia, primarily Indonesia.

## Goals
- Drive OMO X pre-order reservations
- Communicate Embodied AI and self-balancing differentiation
- Build brand credibility and desirability
- Surface dealer locations and drive test ride bookings

## Tech stack
- Next.js 15 App Router
- Tailwind CSS + shadcn/ui
- GSAP + ScrollTrigger
- TypeScript
- Medusa.js or Shopify Storefront API (e-commerce, TBD)
- Sanity.io (CMS for news and product content)
- Vercel (deployment)

## Design system
- Colors: docs/design-system/colors.md

## Design principles
- Clean and minimal — Rivian/Tesla reference
- Dark hero and dark sections, light body sections (#F3F5EE)
- Sharp corners everywhere — no border radius
- Accent (#BAD969) used intentionally — never decoratively
- Product is always the hero — no decorative clutter
- Typography drives hierarchy — not color or decoration

## Target audience
- Tech enthusiasts, early adopters
- Urban riders, Southeast Asia — Indonesia primary market
- Age 25–40, design and tech aware

## Out of scope (v1)
- Checkout flow
- User accounts
- Multilingual (EN only)
- Mobile app deep links
